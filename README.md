- Sempre que for fazer alterações, confira se está na BRANCH correta. Na dúvida, pergunte.

</br>


## Recomendações
- Manter as pastas no computador, não no Pendrive ou algo do tipo. Recomendo criar uma pasta separada na área de trabalho com nome "Github TCC".
- É interessante manter um histórico de pelo menos 2 versões anteriores com a atual. Pra isso, crie cópia da pasta, renomeie a versão, e edite essa nova.

</br>


## Como baixar o repositório:
### 1º vez acessando o projeto:
- Copia o caminho da pasta criada, abre o console e dê CD (caminho da pasta)
- Depois, dê o comando:
```
git clone https://github.com/luizfelipe-gif/TCC-versoes .
```
- Feito isso, mude para a **sua** Branch:
```
git checkout -b NOMEdaBRANCH
```
Feito isso, faça as alterações, e siga pro passo de como enviar.


### Outras vezes - Atualizar o projeto baixado anteriormente:
- Abre o console, dê o CD (link da pasta)
- Dê o comando:
```
git pull origin NOMEdaBRANCH
```
Isso vai trazer o projeto atualizado, porém **se houver alterações, elas serão perdidas.**
